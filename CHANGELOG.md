# Changelog

All notable changes to this project are documented here.

## [v0.2.1] - 2026-03-01

- Added release-helper polish and release-snippet artifact support in release workflow.
- Improved README release visibility links and latest-release badge.
- Updated tap formula to align with v0.2.1 tarball metadata.

## [v0.2.0] - 2026-03-01

- Migrated the CLI from Bash to Rust while preserving command coverage.
- Added CI workflows, GitHub Pages docs workflow, and expanded docs hub.
- Improved release process documentation and Homebrew tap integration.

## [v0.1.0] - 2026-03-01

- Initial public release of `sk` as a macOS Keychain helper CLI.
